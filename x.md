กรุณาอ่านรายละเอียดทุกข้อด้านล่าง ก่อนกรอกข้อมูลแจ้งเบาะแสยาเสพติดตามความเป็นจริง
เพื่อให้เจ้าหน้าที่นำข้อมูลของท่านไปทำงานได้จริง และรวดเร็วที่สุด
* ข้อมูลสำคัญในการแจ้งเบาะแส : ใคร — ทำอะไร — ที่ไหน — อย่างไร — กับใคร — เมื่อไหร่ *

หรือติดต่อสอบถามได้ที่เบอร์สายด่วน 1386 ข้อมูลผู้แจ้งปิดเป็นความลับ

1. รูปพรรณสัณฐานของผู้กระทำผิดให้ชัดเจน เช่น ส่วนสูง สีผิว รูปร่าง ทรงผม ลักษณะเด่น เช่น รอยสัก แผลเป็น สิว ไฝ จัดฟัน ใส่แว่น หัวเถิก หัวล้าน พิการส่วนใดของร่างกาย เป็นต้น
2. ผู้กระทำผิดอาศัยอยู่กับใคร และมีพ่อแม่ญาติพี่น้องชื่ออะไร กรณีทราบแต่ชื่อเล่นผู้กระทำผิด
3. ช่วงเวลาซื้อ-ขาย ช่วงเวลาเสพ-จับกลุ่มมั่วสุม ช่วงเวลาขน-ลำเลียงยาเสพติด พร้อมสถานที่
4. ราคาซื้อ-ขาย แหล่งซื้อ จุดซุกซ่อนยาเสพติดถ้าทราบ
5. ยานพาหนะ และอาวุธปืนของผู้กระทำผิดถ้ามี
6. ข้อมูลพื้นที่ บ้านเลขที่ / ตึกอาคาร / ห้อง / ชั้น / ตรอก / ซอย / ถนน ให้ละเอียด
